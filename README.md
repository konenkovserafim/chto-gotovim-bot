# Что готовим? v6

Telegram-бот для выбора еды для Серафима и Тани.

## Что есть в v6

- нижняя клавиатура Telegram;
- категории: завтрак, обед, ужин, перекус;
- рецепты вынесены в `recipes.json`;
- 40 рецептов: по 10 на каждый раздел;
- более красивые карточки блюд;
- калории отдельно для Серафима и Тани;
- примерные БЖУ;
- ингредиенты и пошаговое приготовление;
- избранное;
- список продуктов.

## Railway

Переменная окружения:

```text
BOT_TOKEN=токен_от_BotFather
```

Запуск:

```bash
python bot.py
```
