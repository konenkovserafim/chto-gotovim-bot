# Что готовим? v8

Telegram-бот с рецептами для Серафима и Тани.

## Что есть

- нижняя клавиатура
- категории: завтрак, обед, ужин, перекус
- рецепты отдельно в `recipes.json`
- красивые карточки рецептов
- избранное
- улучшенный список продуктов:
  - добавление продуктов из рецепта
  - объединение дублей
  - ручное добавление продукта
  - удаление отдельных продуктов
  - очистка списка
