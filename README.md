# Что готовим?

Telegram-бот с меню и рецептами для Серафима и Тани.

## Версия 3.1

- 100 нормальных рецептов: 25 завтраков, 35 обедов, 25 ужинов, 15 перекусов
- Постраничный выбор блюд
- Карточки рецептов
- Избранное
- Список продуктов
- Нижняя клавиатура
