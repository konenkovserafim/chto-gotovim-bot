# Что готовим?

Telegram-бот для выбора блюд и рецептов.

## Версия v5

- рецепты вынесены в отдельный `recipes.json`
- нижняя клавиатура работает
- категории: завтрак, обед, ужин, перекус
- карточки блюд
- избранное
- список продуктов

Для Railway нужна переменная окружения:

```text
BOT_TOKEN=токен_бота
```
