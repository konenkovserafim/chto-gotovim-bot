# Что готовим? v7

Telegram-бот с меню для Серафима и Тани.

## Что есть в v7

- нижняя клавиатура;
- категории: завтрак, обед, ужин, перекус;
- 40 рецептов в `recipes.json`;
- красивые карточки рецептов;
- случайный выбор блюда;
- избранное с сохранением в `user_data.json`;
- список продуктов с сохранением в `user_data.json`;
- очистка избранного и списка продуктов.

## Railway

Переменная окружения:

```text
BOT_TOKEN=токен_от_BotFather
```

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```
